# Overview

**Python Cursus** is a learning path from the **42 Network curriculum (1337 Coding School – UM6P)**.  

It teaches **Python programming** from fundamentals (Modules 00–05) to advanced topics (Module 42) through exercises and projects.  

By the end of the cursus, students will be able to write **structured, modular, and object-oriented Python code**, handle files, exceptions, and external packages.

---

# Modules

### Module 00 – Introduction to Python
- Python basics: syntax, running scripts, printing output  
- Understanding interpreters and IDEs  

### Module 01 – Variables, Types & Input
- Data types: `int`, `float`, `str`, `bool`  
- Variable assignment and type conversion  
- User input handling  

### Module 02 – Control Flow
- Conditional statements: `if`, `elif`, `else`  
- Loops: `for`, `while`  
- Loop control: `break`, `continue`  

### Module 03 – Functions
- Defining and calling functions  
- Parameters, arguments, and return values  
- Variable scope and lifetime  

### Module 04 – Data Structures
- Lists, tuples, dictionaries, and sets  
- Indexing, slicing, and iteration  
- Common methods and operations  

### Module 05 – Classes & OOP
- Classes and objects  
- Attributes and methods  
- Inheritance and polymorphism
